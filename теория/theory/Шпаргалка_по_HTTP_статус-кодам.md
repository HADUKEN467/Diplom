## Информационные (1xx)

- **100 Continue** - сервер получил заголовки, клиент может отправлять тело
    
- **101 Switching Protocols** - сервер переключает протоколы
    
- **102 Processing** - сервер обрабатывает запрос
    

## Успешные (2xx)

- **200 OK** - запрос успешно выполнен
    
- **201 Created** - ресурс создан
    
- **202 Accepted** - запрос принят, но обработка не завершена
    
- **204 No Content** - успешно, но нет содержимого для возврата
    

## Перенаправления (3xx)

- **301 Moved Permanently** - ресурс перемещен навсегда
    
- **302 Found** - временное перенаправление
    
- **304 Not Modified** - контент не изменился (кеширование)
    

## Клиентские ошибки (4xx)

- **400 Bad Request** - неверный синтаксис запроса
    
- **401 Unauthorized** - требуется аутентификация
    
- **403 Forbidden** - доступ запрещен
    
- **404 Not Found** - ресурс не найден
    
- **405 Method Not Allowed** - метод не поддерживается
    
- **409 Conflict** - конфликт с текущим состоянием
    
- **422 Unprocessable Entity** - семантическая ошибка (валидация данных)
    
- **429 Too Many Requests** - слишком много запросов
### Серверные ошибки (5xx)

- ### Проблемы с кодом приложения:

- 500 - Ошибки в логике приложения
    
- 501 - Не реализованный функционал
    

### Проблемы с внешними сервисами:

- 502 - Внешний сервис недоступен
    
- 504 - Таймаут внешнего сервиса
    

### Проблемы с ресурсами:

- 503 - Перегрузка сервера
    
- 507 - Нехватка места
    
- 509 - Превышение лимитов
    

### Проблемы с инфраструктурой:

- 505 - Несовместимость протоколов
    
- 508 - Проблемы конфигурации
    
- 511 - Проблемы аутентификации
    

### Временные проблемы:

- 503 - Техническое обслуживание
    
- 504 - Временные сетевые проблемы
    
- 502 - Временная недоступность сервисов
