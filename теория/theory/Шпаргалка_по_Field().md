### Валидация чисел

- `gt` (greater than) - больше чем
    
- `ge` (greater or equal) - больше или равно
    
- `lt` (less than) - меньше чем
    
- `le` (less or equal) - меньше или равно
    
- `multiple_of` - кратно числу
### Валидация строк

- `min_length` - минимальная длина
    
- `max_length` - максимальная длина
    
- `regex` - регулярное выражение
### Значения по умолчанию

- `default` - статическое значение по умолчанию
### Метаданные и документация

- `description` - описание поля
    
- `example` - пример значения
    
- `title` - заголовок поля
    
- `alias` - альтернативное имя для JSON
